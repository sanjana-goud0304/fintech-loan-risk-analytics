# Project Screenshots

This folder contains SQL analytics outputs and database setup screenshots for the FinTech Loan Risk & Customer Analytics project.

## Folder Structure

### analytics_outputs/
Contains SQL query outputs and KPI analysis screenshots.

### database_setup/
Contains database schema and table structure screenshots.

### sql_queries/
Contains complex SQL query screenshots used for analytics and reporting.

## Key Analytics Covered

- Loan distribution analysis
- Repayment behavior analysis
- High-risk customer identification
- Employment-wise loan segmentation
- Database schema design
